## 起動

### 通常起動（ビルド済み）

```bash
cd mvp && npm start
```

- `http://localhost:4173/viewer.html` でブラウザが自動起動します
- 起動時に `data/rapid-sample.json` を自動生成します
- 停止: `Ctrl+C`

### 初回（ビルドから）

```bash
cd mvp
npm install
npm run build
npm start
```

### レガシー起動（ビルド不要・旧JS直実行）

```bash
node mvp/legacy/start_viewer.js
```

TypeScriptのビルドなしで動作します。機能は同等ですが、TS側の変更は反映されません。

## ビジュアルテスト（Playwright）

```bash
npm run test:visual
```

## ユニットテスト（自動実行向け）

```bash
npm test
```

監視モード（保存ごとに再実行）:

```bash
npm run test:unit:watch
```

CI と同じ実行:

```bash
npm run test:ci
```

スナップショット更新:
```bash
npm run test:visual:update
```

---

## コマンド一覧

| コマンド | 説明 |
|----------|------|
| `npm install` | 依存パッケージのインストール |
| `npm run build` | TypeScript全体をビルド |
| `npm run build:node` | Node側のみビルド |
| `npm run build:browser` | ブラウザ側のみビルド |
| `npm start` | ビューア起動（ビルド済みが前提） |
| `npm test` | ユニットテスト実行 |
| `npm run test:unit:watch` | ユニットテスト監視実行 |
| `npm run test:ci` | CI と同じユニットテスト実行 |
| `npm run test:visual` | Playwright ビジュアルテスト実行 |
| `npm run test:visual:update` | テストスナップショット更新 |
