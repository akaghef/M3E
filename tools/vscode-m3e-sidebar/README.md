# M3E Sidebar

Minimal local VS Code extension that adds an `M3E` icon to the Activity Bar.

## What it does

- `Launch Beta`
- `Update And Launch`
- `PJ03 Demo Full`
- `PJ03 Demo Core`
- `PJ03 Demo Scope`
- `PJ03 Demo Preflight`

Each action opens a VS Code terminal and runs the existing repo command.

## How to try it

1. Open this repo in VS Code.
2. Open `Run and Debug`.
3. Launch `M3E: Sidebar Extension Host`.
4. In the Extension Development Host window, look for the `M3E` icon in the Activity Bar.

This extension is intentionally local and unbundled.
