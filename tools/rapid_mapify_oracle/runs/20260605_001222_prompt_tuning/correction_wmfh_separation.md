# Correction: WMF-H Separation Principle

## Mistake

The previous prompt asked the generation worker to output JSON. That violated the separation principle by mixing the human-facing generation format with the machine delta format.

## Corrected Boundary

- Worker output: WMF-H / MF-H heading fragment only.
- Converter script: `tools/rapid_mapify_oracle/scripts/wmf_h_to_delta.py`.
- Machine delta: appendChildren JSON generated by the converter, not by the worker prompt.

## Verified Pipeline

```text
WMF-H worker output
-> wmf_h_to_delta.py --parent-id n_animals
-> scenario_a_delta_from_wmfh.json
-> evaluate_case.py
-> score 1.0 / passed
```
