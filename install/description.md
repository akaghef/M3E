# M3E — MMME=Mind Map Model Engine

科学研究のための思考支援マインドマップツール。

日常のメモから業務レベルの構造化、研究オントロジーまで——
マインドマップで知識やアイデアを育てるためのツールです。

## インストール

1. `M3E-Setup-v260412.exe` をダウンロード（安全設定により継続を手動でする必要があります）
2. ダブルクリックして実行（安全設定により継続を手動でする必要があります）
3. 画面の案内に従ってインストール
4. デスクトップの M3E アイコンから起動（ブラウザで起動）

## 主な機能

- マインドマップの作成・編集・階層管理
- Flash / Rapid / Deep の 3 バンド切り替え
- FreeMind (.mm) / JSON のインポート・エクスポート
- ピンチズーム（タッチデバイス対応）
- Linear パネルによるスコープメモ
- チュートリアルデータ同梱

## 動作環境

- Windows 10 / 11
- Node.js 22 以上（インストーラーが自動でセットアップ）
- Chrome, Edgeブラウザ
