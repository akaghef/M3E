# specs/

PJv Factory の worker は、PJv* ごとにここへ割り当て仕様書を書く。

命名:

```text
pjvXX_<slug>.md
```

例:

```text
pjv35_local_project_design_report.md
```

仕様書は `instructions/pjv_factory/assignment_spec_template.md` から作る。
