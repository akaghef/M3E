## 特殊キー

| キー | 動作 | 備考 |
|------|------|------|
| `Tab` | 選択ノードの子を追加して編集開始 | depth 方向に展開 |
| `Enter` | 選択ノードの兄弟を追加して編集開始 | breadth 方向に展開 |
| `Alt+Enter` | scope に入る | 選択ノードを scope として設定 |
| `F2` | 選択ノードのテキスト編集を開始 | |
| `Space` | 選択ノードの折り畳み／展開をトグル | 複数選択中は全選択ノードに適用 |
| `Delete` | 選択ノードの部分木を削除 | 複数選択中は selection root をすべて削除 |
| `Backspace` | 選択ノードの部分木を削除 | scope root を選択中かつ scope 履歴あり → 一段上に戻る |
| `Escape` | カット状態をキャンセル / その他コンテキスト依存のキャンセル | |

## 矢印キー

| キー | 動作 | 備考 |
|------|------|------|
| `↑` | breadth 方向（兄弟）で1つ前のノードへ | |
| `↓` | breadth 方向（兄弟）で1つ後のノードへ | |
| `←` | depth 方向で親ノードへ | scope root にいる場合は scope を一段上に戻る |
| `→` | depth 方向で第一子ノードへ | folder ノードの場合は scope に入る |
| `Shift+↑` | 選択範囲を breadth 方向に1つ上へ拡張 | |
| `Shift+↓` | 選択範囲を breadth 方向に1つ下へ拡張 | |
| `Shift+←` | — | |
| `Shift+→` | — | |

## マウス・ポインター操作

| 操作 | 動作 | 備考 |
|------|------|------|
| クリック | ノードを単一選択 | アンカーをリセット |
| `Ctrl/Cmd` + クリック | ノードを選択にトグル追加／除外 | アンカーをクリックしたノードに更新 |
| `Shift` + クリック | アンカー〜クリックしたノードの範囲を選択 | `visibleOrder`（表示順）で範囲を決定 |
| ダブルクリック | 子なし → テキスト編集開始 / 子あり → scope に入る | |
| ノードをドラッグ → ドロップ | ドロップ先の子として reparent | |
| 背景をドラッグ | キャンバスをパン | |
| `Ctrl/Cmd` + ホイール | ズームイン／アウト | ポインター位置を中心にズーム |
| ホイール（縦） | 上下スクロール（pan Y） | |
| ホイール（横） | 左右スクロール（pan X） | |

## インライン編集中のキー

インライン編集（`F2` / `Tab` / `Enter` で開始後）は通常モードとは別のキーマップになる。

| キー | 動作 |
|------|------|
| `Enter` | 編集を確定 |
| `Escape` | 編集をキャンセル（変更破棄） |
| `Shift+Enter` | 改行を挿入（テキスト内改行） |
| その他文字キー | ブラウザ標準のテキスト入力（Ctrl+C/V/Z なども通常通り動作） |
