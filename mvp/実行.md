# MVP 立ち上げ

## 初回（ビルドが必要）

```bash
cd mvp
npm install
npm run build
npm start
```

## 2回目以降（ビルド済み）

```bash
cd mvp && npm start
```

## レガシー（TypeScript ビルド不要・旧 JS 直実行）

```bash
node mvp/start_viewer.js
```


# Visual Test
```
npm --prefix mvp run test:visual

# run MVP(js)
```
node mvp/start_viewer.js
