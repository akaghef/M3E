基本構成は次になります。

センサー基板  
→ Bluetooth  
→ スマホ（ゲートウェイ）  
→ WebSocket / HTTP  
→ サーバ  
→ Web UI / 配信

スマホは BLE 受信して JSON を送るだけの役割になります。ここでは状態管理や地図描画は行いません。単なる **edge gateway** です。

その先はストリーミングになります。代表的な形は次です。

スマホ → WebSocket → サーバ  
サーバ → WebSocket → 複数クライアント

この形にすると

・複数ブラウザで同時表示  
・遠隔監視  
・ログ保存  
・リプレイ  
・ライブ配信

がそのまま実装できます。
