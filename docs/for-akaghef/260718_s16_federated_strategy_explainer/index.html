このバンドルについて

  • これは単一ファイルではなく index.html + views/ のフォルダ一式です。
  • 右ペインの iframe は views/ 内の個別 HTML を読み込みます。タブでビューを切り替えられます。
  • 左ペイン = 別のビューを「追加」してもらうためのプロンプト集（チャットUIではありません）。

使い方

  • 右上の 🌙 ダーク / ☀ ライト でテーマを切り替え。
  • テンプレートの プロンプトをコピー を押し、ローカルファイルを読めるAIに貼り付ける。
  • 返ってきた iframe HTML を views/ に保存し、ビルドを再実行すると新しいタブとして増えます。

開き方の注意

  • Chrome/Edge は file:// での iframe ローカル読み込みをブロックします。
  • Firefox で開くか、フォルダ内で簡易サーバを起動してください（内容は外部通信しません）。

安全性

  • iframe は sandbox="allow-scripts"。親 DOM へはアクセスしません。
  • ネットワーク通信・外部 CDN・ストレージ利用はありません。
