# M3E アプリ機能ギャップ列挙

Home 画面実装後の会話で出た「アプリとして普通あるべきで、M3E に不足 or 弱いもの」の一括列挙。
一部は既存実装あり（例: audit ログ、暗号化ポリシー）だが UI/導線がない or 体系化されていないものも含む。
別途タスク化する際の source。

## ユーザー周り
- アカウント / 認証（ローカル専用 or クラウド同期時の認証フロー）
- 複数プロファイル / ワークスペース切り替え
- 設定画面（テーマ / フォント / キーバインド / 言語 / 保存先 / AI キー管理を集約）
- オンボーディング / チュートリアル（初回起動時のウェルカム、空状態の誘導）

## ドキュメント管理
- ごみ箱（delete の一次保管、復元可能）
- バージョン履歴 / タイムマシン（node-level ではなく doc スナップショット）
- エクスポート（Markdown / JSON / PDF / OPML）と Import 画面の網羅
- 自動バックアップ（日次 .zip など）
- テンプレート（"会議議事録" "研究ログ" など雛形から新規作成）

## 検索・ナビゲーション
- グローバル検索（全 doc 横断、Cmd+K）
- コマンドパレット（全アクションをキーボードで）
- 最近開いた履歴 / パンくず / 戻る進む

## 共有・連携
- 共有リンク / read-only 公開
- コメント / メンション（協業時）
- 通知センター（同期エラー、コメント、招待）
- Obsidian 以外の外部連携（Notion / Google Drive / Slack 送信）

## AI / 自動化
- AI チャットサイドバー（node を文脈に質問）
- 自動タグ / 自動要約
- ショートカット・マクロ（一連の操作を記録・再生）

## 運用・信頼性
- オフラインモード / 同期ステータス UI
- エラーレポート / クラッシュリカバリ（起動時の「前回の状態を復元しますか？」）
- ヘルプ / ドキュメント / ショートカット一覧（`?` キーで cheatsheet）
- アップデート通知（新バージョン availability）
- 利用状況ダッシュボード（doc数、容量、同期状況）

## アクセシビリティ・基本 UX
- キーボードフルアクセス（マウスなし操作）
- ダークモード / テーマ切替
- 多言語 i18n（UI 文言の英日混在を体系化）
- Undo/Redo の視認性（履歴スタック閲覧）
- モバイル / タブレット対応

## セキュリティ・プライバシー
- エンドツーエンド暗号化の設定 UI
- ローカルデータ暗号化（パスフレーズロック）
- 監査ログ閲覧 UI（audit 実体はあるはず）

## M3E 特有
- scope ブックマーク / ピン（doc 内の特定 scope をお気に入り）
- グラフ全体ビュー（doc 間リンクを可視化、Obsidian の Graph view 相当）
- alias 参照の一覧 / リンク切れ検出
